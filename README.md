# structured-light-education
Educational visualization of structured light and optical phase patterns.
## Contents

- `phase_basics.py`  
  Basic optical phase visualization using `phase = arctan2(y, x)` on a 2D grid.

## How to Run

1. Clone this repository or download the source.
2. Install Python 3 and Matplotlib (`pip install matplotlib numpy`).
3. Run:

```bash
python phase_basics.py
